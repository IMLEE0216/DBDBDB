# MySQL Join

- 데이터베이스 내의 여러 테이블에서 가져온 데이터를 조합하여 -> 하나의 테이블로 표현
- SELECT와 조합해서 쓸수있다.

## 1. Inner Join

- ON 과 같이 사용되며, ON절의 조건과 만족하는 데이터만 가져온다.
- ON에는 WHERE 절 쓰듯이 쓰면된다.
  - table name1 **INNER JOIN** table name2 **ON** 조건
  - table name1 **JOIN** table name2 **ON** 조건
